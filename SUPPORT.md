## Common Issues

### Annotations not appearing

1. Check that the file is saved
2. Try refreshing the sidebar (click refresh icon)
3. Verify VS Code version is 1.105.0 or higher

### Export not working

1. Ensure you have annotations in the current workspace
2. Check that clipboard access is permitted
3. Try exporting to a document instead

### Webview not rendering

1. Reload VS Code window (Ctrl+R / Cmd+R)
2. Check for errors in Developer Tools (Help > Toggle Developer Tools)
3. Update to the latest extension version

### Storage location unclear

Run "Annotative: Storage Info" command to see where annotations are stored.
